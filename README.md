## Pre-Interview-Test

## Install

`npm install pre-interview-test -g`

but if you clone the project

`npm install`

`sudo npm link`



## Usage and Example

	$ pre-interview-test batman
	Title: Batman
	Year: 1989

	$ pre-interview-test avengers
	Title: Avengers
	Year: 1987

	$ pre-interview-test ultron
	Title: Avengers: Age of Ultron
	Year: 2015
