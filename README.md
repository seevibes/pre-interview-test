# Seevibes

This gem enables you to find a movie by a partial name on Imdb

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'seevibes'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ seevibes-1.0.0.gem

## Usage

seevibes search batman
